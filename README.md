# pdf_ai_chatbot


This project is an intelligent PDF Question-Answering Chatbot built using Python, Gradio, LangChain, and Large Language Models (LLMs). Users can upload one or more PDF documents, and the system processes the content using text chunking, embeddings, and vector search to retrieve relevant information. By leveraging Retrieval-Augmented Generation (RAG), the chatbot provides accurate, context-aware answers based on the uploaded documents. The application features an intuitive Gradio interface, making document exploration, knowledge extraction, and conversational querying simple and efficient.

Key Features

=> Upload and chat with PDF documents
=> Retrieval-Augmented Generation (RAG)
=> Semantic search using vector embeddings
=> Context-aware question answering
=> User-friendly Gradio web interface
=> Support for multiple document queries
=> Fast and efficient document retrieval and response generation
